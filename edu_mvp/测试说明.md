# 自动化测试脚本

## 安装依赖

```bash
cd ~/Desktop/edu_agent_system/edu_mvp/frontend
npm install --save-dev jest @testing-library/react @testing-library/jest-dom @testing-library/user-event jest-environment-jsdom
```

## 运行测试

### 运行所有测试
```bash
npm test
```

### 运行单个测试文件
```bash
npm test -- StudentHome.test.jsx
npm test -- Quiz.test.jsx
npm test -- WrongBook.test.jsx
npm test -- AdminHome.test.jsx
npm test -- AdminQuestions.test.jsx
```

### 监视模式（文件变更后自动重新测试）
```bash
npm test -- --watch
```

## 测试覆盖范围

### ✅ 学生端功能
- **StudentHome.test.jsx** - 首页加载、章节展示、错误处理
- **Quiz.test.jsx** - 题目加载、答题流程、错题记录
- **WrongBook.test.jsx** - 错题加载、清空功能

### ✅ 管理台功能
- **AdminHome.test.jsx** - 数据统计、Tab 切换、表格展示
- **AdminQuestions.test.jsx** - 题目筛选、编辑功能

## 注意事项

1. **所有 API 调用都已 Mock** - 测试不会真实调用后端
2. **localStorage 已 Mock** - 错题本功能可测试
3. **react-router-dom 已 Mock** - 路由跳转可测试

运行测试后，会显示：
- ✅ 通过的测试用例数
- ❌ 失败的测试用例数
- 📊 代码覆盖率（如果配置了 --coverage）
