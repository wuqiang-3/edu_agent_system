## ⚠️ 待完善 / 已知局限

- **本初中学数学学习系统未经真实教学场景验证**，题目质量、难度分级、自适应推荐算法均未经过学生/教师试用反馈
- **`mvp_main.py` 的 Agent 编排流程完全无自动化测试**，依赖 DeepSeek API 实时调用，受限于 API 可用性
- **`hash()` 作为随机种子不可靠**（Python 3.x 中 PYTHONHASHSEED 导致跨进程不一致），影响题目生成的确定性
- **安全审计为代码静态分析**，未执行实际的渗透测试或动态扫描
- **CORS + credentials 组合在浏览器中实际不会生效**（浏览器拒绝），但安全官报告的"非浏览器客户端绕过"风险在公网部署场景下仍成立
- 系统当前未有真实用户数据，性能基准和压力测试无法进行
